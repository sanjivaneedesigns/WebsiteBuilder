# My Element Library

A personal collection of reusable UI elements, components, and patterns for use across website builder projects.

## Overview

This library serves as a custom element repository where you can store, document, and reuse your own design and code components.

## Structure

Add your custom elements organized by type:

- **Components** – Buttons, cards, forms, navigation elements
- **Sections** – Hero blocks, feature grids, testimonial layouts
- **Templates** – Full page layout starters
- **Utilities** – Helper styles, mixins, tokens

## Usage

Reference elements from this library when building new projects to ensure consistency and accelerate development.
