# Section Library Framework

This framework documents the reusable page section patterns available for website construction.

## Overview

Sections are the building blocks of every page. This library catalogs available section types, their use cases, and composition rules.

## Section Categories

- **Hero Sections** – Full-width openers, split layouts, video backgrounds
- **Feature Sections** – Grid, list, icon-based feature showcases
- **Social Proof** – Testimonials, logos, case studies, stats
- **Content Sections** – Long-form text, editorial layouts, FAQs
- **CTA Sections** – Banner CTAs, inline CTAs, sticky bars
- **Footer Sections** – Standard, minimal, mega footers
