# Icon Libraries

This document specifies the icon libraries and usage guidelines for the design system.

## Overview

Icons provide visual clarity and reinforce meaning in UI components. Use icons consistently and accessibly.

## Recommended Libraries

- **Lucide** – Clean, consistent open-source icon set (recommended default)
- **Heroicons** – Tailwind-compatible, two styles: outline and solid
- **Phosphor Icons** – Flexible, multi-weight icon family
- **Tabler Icons** – Large set of outline icons, great for dashboards
- **Custom Icons** – Project-specific SVG icons stored in `/assets/icons/`

## Usage Rules

- Always provide `aria-label` or `aria-hidden` on icon elements
- Use consistent sizing: 16px (sm), 20px (md), 24px (lg), 32px (xl)
- Match icon weight/style to typography weight in context
