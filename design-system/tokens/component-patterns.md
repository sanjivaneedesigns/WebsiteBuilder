# Component Patterns

This document defines the reusable UI component patterns and their token specifications.

## Overview

Component patterns provide consistent building blocks for assembling page sections and layouts.

## Core Components

- **Buttons** – Primary, secondary, ghost, icon, destructive
- **Cards** – Standard, feature, testimonial, pricing, blog
- **Forms** – Input, textarea, select, checkbox, radio, toggle
- **Navigation** – Top nav, sidebar, mobile menu, breadcrumbs, tabs
- **Modals & Overlays** – Dialog, drawer, tooltip, popover
- **Media** – Image, video, avatar, icon, badge
