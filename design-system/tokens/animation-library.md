# Animation Library

This document catalogs the animation tokens and motion design patterns used across the design system.

## Overview

Consistent, purposeful animation enhances user experience without distraction. Use these tokens to apply motion with intent.

## Token Categories

- **Duration** – fast (100ms), normal (250ms), slow (400ms), xslow (600ms)
- **Easing** – ease-in, ease-out, ease-in-out, spring, bounce
- **Entrance Animations** – fade-in, slide-up, slide-in, scale-in
- **Exit Animations** – fade-out, slide-down, scale-out
- **Scroll Animations** – parallax, reveal-on-scroll, stagger
